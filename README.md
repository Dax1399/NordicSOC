# Nordic SOC — AI-Driven SOC website

Marketing homepage for **Nordic SOC**, the EU-sovereign, AI-driven managed Security Operations Centre operated from Stockholm (part of the G'Secure Labs & Gateway Digital group).

## Structure
- `index.html` — the complete, self-contained homepage (HTML + CSS + a little vanilla JS). No build step.

## Sections
Hero → The Nordic AI-SOC Framework (6-layer lifecycle) → AI SOC Platform · AI Investigation · Agentic Security Operations · Compliance & GRC (each with its own dashboard) → Proof metrics → Stockholm Centre → SOC Architectures → Full-Stack Services → Differentiators → Network of Trust → Contact.

## Deployment
Static site, deployed on Vercel. No framework or build command required — Vercel serves `index.html` from the repo root.
